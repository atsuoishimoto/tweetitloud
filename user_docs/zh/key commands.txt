NVDA 按鍵指令列表

以下許多按鍵指令都必須使用 NVDA 的組合鍵， NVDA 的組合鍵即是小鍵盤的數字鍵 0，又或一般所說的插入鍵。你也可把大寫鎖鍵設為 NVDA 的組合鍵。

通用指令

Control: 暫停朗讀
Shift: 繼續朗讀

NVDA+1: 鍵盤說明模式開關
在鍵盤說明模式開啟之後，你可按下任何按鍵組合，如果這個案件組合是 NVDA 的指令，它便會告訴你按鍵的用途，要關閉鍵盤說明模式，同樣可按 NVDA+1。

NVDA+t: 視窗標題
報出目前應用視窗的標題，連案兩次會把標題拼讀出來，連按三次會把標題複製到剪貼簿。

NVDA+b: 前景視窗內容
報出目前視窗內容。如果程式第一次報讀對話盒內容的時候你錯過了，你可利用這個指令讓它重讀對話盒內容。

NVDA+n: 打開 NVDA 功能表
你可利用這個按鍵指令打開 NVDA 的功能表，然後搞定一些程式的選項設定。

NVDA+q: 退出程式
按鍵後會問你是否要退出 NVDA，按「是」會立即退出程式，按「否」就不會退出。

NVDA+s: 發音模式
每次按鍵程式都會在講話、發出逼逼聲和靜音三個模式之間切換。

NVDA+F12: 日期和時間
按鍵後會報出目前時間，連按兩次則報出目前日期。

NVDA+End: 前往程式狀態列
按鍵後程式會嘗試尋找目前應用程式的狀態列，如果找到就把瀏覽目標移到並報出當中的內容。

NVDA+f: 格式資訊
在目前的文件中，報出現時所在位置和格式資訊。

Control+NVDA+f1: 應用程式所屬檔名
報出目前應用程式的執行檔名稱，也會讀出現時 NVDA 載入的模組。

NVDA+Shift+b: 電池狀態
報出系統目前的電量和是否接通交流電源等資料。

NVDA+5: 報告動態內容開關
切換是否報出動態內容改變，可在「命令提示字元」模式(或稱為 DOS 模式)的視窗中發生作用。

NVDA+6: 瀏覽游標跟隨打字游標開關
切換是否讓瀏覽游標跟隨打字游標走。

NVDA+7: 瀏覽目標跟隨系統焦點開關
切換是否讓瀏覽目標跟隨系統焦點走。

NVDA+Tab: 目前焦點項目
報出目前系統焦點的項目。

NVDA+f2: 不理會下次按鍵
按鍵後隨即按其他按鍵 NVDA 就不理會後者，讓後者能夠直接控制作業系統或目前應用程式。

Shift+NVDA+向上鍵: 朗讀目前已選取的內容
在文字編輯區中，報出目前選取了的文字，如果沒有選取文字它也會有所表示。

NVDA+向上鍵: 朗讀目前一行
在文字編輯區內報出目前的一行文字。

NVDA+向下鍵: 全部朗讀
在文字編輯區或文件中，從目前游標所在的一行文字開始朗讀直至結尾。

NVDA+f1: 物件的進佳資訊
報出目前項目的進佳資訊，這些資訊對於程式的開發者才較為有用。

Control+NVDA+c: 儲存程式設定
儲存目前 NVDA 的程式設定。

Control+NVDA+r: 套用已經儲存的設定
按鍵後程式會放棄目前的設定，套用已經儲存的設定。

更改設定按鍵
下面的按鍵能夠更改程式的選項設定(這些設定也可在程式的設定對話方塊中找到)

NVDA+2: 邊打邊唸字元開關
切換是否把輸入的字元報讀出來。

NVDA+3: 邊打邊唸自詞開關
切換是否把輸入的字詞報讀出來。

NVDA+4: 邊打邊唸按鍵開關
切換是否把輸入的所有按鍵也都報讀出來。

NVDA+p: 朗讀標點開關
切換每當程式報讀的時候，是否把標點符號也都報讀出來。

NVDA+m: 鼠標移動通知開關
切換每當鼠標移動的時候，是否報出它所指的物件或字詞。

NVDA+u: 進度列音效切換
切換每當進度列改變時的表示方式，表示方式有四個，包括發出逼逼聲、每隔百分之十就說出來、兩者皆同時開啟以及兩者皆同時關閉。

Control+NVDA+向左鍵: 上一個語音選項
切換到上一個語音選項。
 
Control+NVDA+向右鍵: 下一個語音選項
切換到下一個語音選項。

Control+NVDA+向上鍵: 增加語音設定值
增加目前語音選項的設定值。

Control+NVDA+向下鍵: 減少語音設定值
減少目前語音選項的設定值。

打開對話方塊按鍵
下面的按鍵可用來叫出 NVDA 的設定對話方塊

Control+NVDA+g: 打開一般設定對話方塊
顯示一般設定對話方塊。

Control+NVDA+s: 打開語音引擎設定對話方塊
顯示語音引擎選擇對話方塊。

Control+NVDA+v: 打開語音設定對話方塊
顯示語音設定對話方塊。

Control+NVDA+k: 打開鍵盤設定對話方塊
顯示鍵盤設定對話方塊。

Control+NVDA+m: 打開滑鼠設定對話方塊
顯示滑鼠設定對話方塊。

Control+NVDA+o: 打開物件說明設定對話方塊
顯示物件說明設定對話方塊。

Control+NVDA+b: 打開虛擬援充設定對話方塊
顯示虛擬援充設定對話方塊。

Control+NVDA+d: 打開文件說明設定對話方塊
顯示文件說明設定對話方塊。

Control+NVDA+z: 打開 Python 主控台
顯示程式的 Python 主控台。

瀏覽目標按鍵
下面的按鍵可用來移動瀏覽目標或報出它所指的物件

NVDA+小鍵盤5: 目前物件
報出目前物件的資訊，連按兩次則把這些資訊拼讀出來，連按三次則把物件的名稱和數值也都複製到剪貼簿。

Shift+NVDA+小鍵盤5: 目前物件的大小
報出目前物件的大小以及所在位置。
 
Control+Shift+NVDA+小鍵盤5: 目前物件所屬
從目前項目開始，讀出所有上一層級的物件，以說明目前物件的位置。

NVDA+小鍵盤8: 瀏覽目標往上一層
把瀏覽目標移到並讀出目前項目上一層級的物件。

NVDA+小鍵盤2: 瀏覽目標往下一層
把瀏覽目標移到並讀出目前項目下一層級的物件。

NVDA+小鍵盤4: 瀏覽目標往上一個
把瀏覽目標移到並讀出目前項目同一層級的上一個物件。

Shift+NVDA+小鍵盤4: 瀏覽目標往上一個或上一層
把瀏覽目標移到並讀出目前項目的上一個物件，移動範圍不限同一層級。

NVDA+小鍵盤6: 瀏覽目標往下一個
把瀏覽目標移到並讀出目前項目同一層級的下一個物件。

Shift+NVDA+小鍵盤6: 瀏覽目標往下一個或下一層
把瀏覽目標移到並讀出目前項目的下一個物件，移動範圍不限同一層級。

NVDA+小鍵盤Enter: 執行物件預設動作
按鍵後會執行目前物件的預設動作，例如目前的物件是按鈕就按下，是清單項目就選取該項目等。

NVDA+小鍵盤除號: 鼠標移到瀏覽目標
把鼠標移到目前的物件。
 
NVDA+小鍵盤乘號: 瀏覽目標移到鼠標
把瀏覽目標移到鼠標所指的物件。

小鍵盤減號: 瀏覽游標移到打字游標
把瀏覽游標移到打字游標的所在位置。

Control+小鍵盤減號: 打字游標移到瀏覽游標
把打字游標移到目前瀏覽游標的所在位置。

NVDA+小鍵盤減號: 瀏覽目標移到系統焦點
把瀏覽目標移到系統焦點。

小鍵盤除號: 滑鼠阻擊
在鼠標所在位置點擊滑鼠左鍵一次。

小鍵盤乘號: 滑鼠右擊
在鼠標所在位置點擊滑鼠右鍵一次。

小鍵盤加號: 瀏覽游標全部朗讀
利用瀏覽游標朗讀目前物件之內的文字，從目前游標的所在位置開始朗讀，直至可見文字的結尾。

NVDA+小鍵盤加號: 瀏覽目標的所有物件
從目前項目開始報出流程之內的所有物件。

物件之內的操作按鍵
下面的按鍵只可在目前物件的範圍內有效

小鍵盤1: 瀏覽游標往上一個字元
把瀏覽游標移到並讀出上一個字元。

Shift+小鍵盤1: 瀏覽游標往行頭
把瀏覽游標移到目前這一行的開頭位置。

小鍵盤2: 報讀瀏覽游標的字元
報出瀏覽游標所在的一個字元。

小鍵盤3: 瀏覽游標往下一個字元
把瀏覽游標移到並讀出下一個字元。

Shift+小鍵盤3: 瀏覽游標往行尾
把瀏覽游標移到目前這一行的結尾。

小鍵盤4: 瀏覽游標往上一個字詞
把瀏覽游標移到並讀出上一個字詞。

小鍵盤5: 讀出瀏覽游標的字詞
報出瀏覽游標所在的一個字詞。

小鍵盤6: 瀏覽游標往下一個字詞
把瀏覽游標移到並讀出下一個字詞。

小鍵盤7: 瀏覽游標往上一行
把瀏覽游標移到並讀出上一行文字。

Shift+小鍵盤7: 瀏覽游標往最先一行
把瀏覽游標移到並讀出最先的一行文字。
 
小鍵盤8: 報出瀏覽游標的一行
報出瀏覽游標所在的一行文字。

小鍵盤9: 瀏覽游標往下一行
把瀏覽游標移到並讀出下一行文字。

Shift+小鍵盤9: 瀏覽游標往最後一行
把瀏覽游標移到並讀出最後一行文字。

NVDA+f9: 訂定複製文字的開始位置
可在複製文字的起點利用這個按件指令訂定一個開始位置。

NVDA+f10: 複製文字到剪貼簿
在同一物件內，可利用這個按件指令，把開始位置和目前瀏覽游標位置之間的文字複製到剪貼簿。

NVDA+c: 朗讀剪貼簿的文字內容
報出剪貼簿內的文字內容，如果剪貼簿的文字太多雖然程式不會一一都說出來，但會讓你知道共有多少個字元。

虛擬援充按鍵
這些按鍵都可在 Internet Explorer、 Firefox 3 以及某些情況才有作用

NVDA+空格鍵: 瀏覽或打字模式的切換
在瀏覽模式和打字模式兩者之間切換。

escape: 切換至瀏覽模式
從打字模式切換至瀏覽模式。

control+NVDA+f: 尋找字串
在目前頁面尋找文字，按鍵之後可打入要找尋的字串，然後再按 enter 鍵。

NVDA+f3: 往下搜尋同樣的文字
向目前頁面底部找尋上一次輸入的字串。

shift+NVDA+f3: 往上搜尋同樣的文字
向目前頁面的頂部找尋上一次輸入的字串。

NVDA+f7: 打開物件列表對話方塊
按鍵後程式會顯示一個對話框，裡面可列出目前頁面的標題、連結和地標等項目。

NVDA+f5: 虛擬援充更新
把程式虛擬援充區的資料更新以載入較更新的頁面資料。
