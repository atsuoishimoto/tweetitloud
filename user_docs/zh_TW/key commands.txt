NVDA 快速鍵指令列表

以下許多按鍵指令都必須結合使用一個 NVDA 鍵， NVDA 鍵是指數字鍵盤上的數字鍵 0（也是Insert鍵），或延伸鍵盤上的插入鍵（Insert鍵）。你也可把大寫鎖定鍵（CAP LOCK）設為 NVDA 鍵。

通用指令

Control: 停止報讀
Shift: 暫停/繼續報讀

NVDA+1: 按鍵指令說明開關
啟動此按鍵指令說明模式後，接著按下任何按鍵組合，如果這個按鍵組合是 NVDA 的指令，即會報讀出此按鍵的功能，但電腦並不執行此指令。若要關閉按鍵說明模式，則再按一次 NVDA+1。

NVDA+T: 讀出視窗標題
讀出目前作用中的應用視窗標題，若連按兩次 T 會將標題拼讀出來，連按三次T 則將標題複製到剪貼簿。

NVDA+b: 讀出前景視窗內容
如果程式第一次報讀對話方塊內容，你錯過了某些訊息，可利用這個指令讓它重讀對話方塊內容。

NVDA+n: 開啟 NVDA 使用者介面功能表
利用這個按鍵指令開啟 NVDA 的功能表，接著可改變一些控制程式的設定選項。

NVDA+q: 退出程式
按鍵後會問你是否要退出 NVDA，按「是」會立即退出程式，按「否」就不會退出。也可將其設定使其直接退出，不會跳出此確認的對話方塊。

NVDA+s: 變更發音模式
每按鍵一次，程式會在報讀、靜音及發出逼逼聲三個模式間依序循環切換。

NVDA+F12: 讀出日期或時間
按鍵後會讀出目前時間，若連按兩次 F12 則讀出目前日期。

NVDA+End: 讀出程式狀態列的訊息
按鍵後程式會嘗試尋找目前應用程式的狀態列，如果有的話，就讀出其內容。

NVDA+f: 讀出格式的資訊
在開啟的文件中，讀出目前游標所在位置字元的格式資訊。

NVDA+Control+F1: 讀出開啟此視窗的應用程式檔名
讀出目前啟動的應用程式檔案名稱，也會讀出現時 NVDA 載入的模組名稱。

NVDA+Shift+b: 讀出電池狀態
適用在筆記型電腦，讀出系統目前電池的可用電量和是否接通交流電源。

NVDA+5: 讀出動態內容開關
切換是否讀出文件內容的改變，例如在命令提示字元視窗(或稱為 DOS 視窗)，讀出新輸入指令後所顯示的內容。

NVDA+6: 瀏覽游標跟隨編輯游標開關
切換是否讓瀏覽游標跟隨編輯游標移動。

NVDA+7: 瀏覽游標跟隨系統焦點開關
切換是否讓瀏覽游標跟隨系統焦點移動。

NVDA+Tab: 讀出目前焦點
讀出目前焦點所在的物件。

NVDA+F2: 忽略下一個NVDA按鍵指令
按鍵後隨即按下其他NVDA指令鍵，則此指令於NVDA中無任何作用，如果此指令有效的話，會直接執行其對應的作業系統指令。

NVDA+Shift+向上鍵: 讀出選取的內容
在可編輯方塊或文件中，讀出目前已選取的文字。

NVDA+向上鍵: 讀出游標所在行的內容
在可編輯方塊或文件中讀出游標所在行的文字，或讀出焦點所在項目。

NVDA+向下鍵: 向下讀出全部
在可編輯方塊或文件中，從目前游標所在的位置往下讀出直到結尾。

NVDA+F1: 讀出物件的測試導航資訊
讀出目前物件的導航資訊，這些資訊是供程式開發者使用。

NVDA+Control+c: 儲存程式設定
儲存 NVDA 所做設定的變更。

NVDA+Control+r: 還原設定
還原至前一次的設定。

變更設定的按鍵指令
下面的按鍵可快速變更程式選項的設定，這些變更也可以從開啟 NVDA 功能表的偏好設定對話方塊中進行變更。

NVDA+2: 讀出輸入字元開關
切換是否將正在輸入的字元報讀出來。

NVDA+3: 讀出輸入字詞開關
切換是否將正在輸入的字詞報讀出來，若是英文則以英文字加上空白為讀出的一個單元，而中文則以標點符號為一個單元。
 
NVDA+4: 讀出輸入指令鍵的開關
切換是否將所按下的任何按鍵報讀出來。

NVDA+p: 讀出標點符號開關
切換是否將標點符號報讀出來。

NVDA+m: 鼠標移動報讀開關
切換每當鼠標移動時，是否讀出它所指的物件或字詞。若要進一步設定可以開啟偏好設定，變更指標所指位置的報讀範圍，包括字元						、字詞、一行或段落等。

NVDA+u: 進度列宣告方式切換
切換每當進度列改變時的報讀方式，報讀方式有四個，依序是：每隔百分之十讀出、發出逼逼聲（此為預設）、兩者皆開啟以及沒有。

NVDA+Control+向左鍵: 移至語音選項內的上一個控制項
直接移至語音選項內的上一個控制項，可選擇的包括人聲、速度、音調及音量等4項。
 
NVDA+Control+向右鍵: 移至語音選項內的下一個控制項
移至語音選項內的下一個控制項。

NVDA+Control+向上鍵: 增加語音設定值
增加目前語音選項內控制項的設定值。

NVDA+Control+向下鍵: 減少語音設定值
減少目前語音選項內控制項的設定值。

顯示設定對話方塊
下面的按鍵可開啟 NVDA 偏好設定功能表內的子選單對話方塊

NVDA+Control+g: 開啟一般設定對話方塊
開啟並顯示一般設定子選單的對話方塊。

NVDA+Control+s: 開啟語音合成軟體設定的對話方塊
開啟並顯示語音合成軟體選項的對話方塊。

NVDA+Control+v: 開啟人聲設定的對話方塊
開啟並顯示人聲設定的對話方塊。

NVDA+Control+k: 開啟鍵盤設定的對話方塊
開啟並顯示鍵盤設定的對話方塊。

NVDA+Control+m: 開啟滑鼠設定的對話方塊
開啟並顯示滑鼠設定的對話方塊。

NVDA+Control+o: 開啟物件說明設定的對話方塊
開啟並顯示物件說明設定的對話方塊。

NVDA+Control+b: 開啟虛擬緩充設定的對話方塊
開啟並顯示虛擬緩充設定的對話方塊。

NVDA+Control+d: 開啟文件格式設定的對話方塊
開啟並顯示文件格式設定的對話方塊。

NVDA+Control+z: 開啟 Python 主控台
開啟並顯示程式的 Python 主控台。

瀏覽物件

NVDA+數字鍵盤5: 讀出目前的物件
讀出目前的物件，連按兩次拼讀出此物件，連按三次則把物件的名稱和數值也都複製到剪貼簿。

NVDA+數字鍵盤 Delete: 讀出目前物件的大小
讀出目前物件的大小以及所在位置。
 
NVDA+數字鍵盤8: 瀏覽至上一層級的物件
將瀏覽目標移到目前項目的上一層級的物件（也就是包含原物件的母物件）並將其讀出來。

NVDA+數字鍵盤2: 瀏覽至下一層級物件
將瀏覽目標移到目前項目下一層級的第一個物件（也就是其子物件的第一個項目）並將其讀出來。

NVDA+數字鍵盤4: 瀏覽至同一層級的上一個物件
將瀏覽目標移到與目前物件同一層級的上一個物件並將其讀出來。

NVDA+Shift+數字鍵盤4: 瀏覽至上一個物件
將瀏覽目標移到與目前物件同一層級或跨層級的上一個物件（也就是說，移動範圍不限同一層級）並將其讀出來。

NVDA+數字鍵盤6: 瀏覽至同一層級的下一個物件
將瀏覽目標移到與目前物件同一層級的下一個物件並將其讀出來。

NVDA+Shift+數字鍵盤6: 瀏覽至下一個物件
將瀏覽目標移到與目前物件的下一個物件（移動範圍不限同一層級）並將其讀出來。

NVDA+數字鍵盤Enter: 執行物件預設動作
按鍵後會執行目前物件的預設動作，例如目前的物件是按鈕就按下，是清單項目就選取該項目等。

NVDA+數字鍵盤除號: 鼠標移到瀏覽物件
將鼠標移到目前的瀏覽物件。
 
NVDA+數字鍵盤乘號: 瀏覽游標移到鼠標
將瀏覽游標移到鼠標所指的物件。

數字鍵盤減號: 瀏覽游標移到編輯游標
將瀏覽游標移到編輯游標的所在位置。

Control+數字鍵盤減號: 編輯游標移到瀏覽游標。  
將編輯游標移到目前瀏覽游標的所在位置，並讀出

NVDA+數字鍵盤減號: 瀏覽游標移到系統焦點
將瀏覽游標移到目前系統焦點的物件。

數字鍵盤除號: 滑鼠左擊
在鼠標所在位置點擊滑鼠左鍵一次。

數字鍵盤乘號: 滑鼠右擊
在鼠標所在位置點擊滑鼠右鍵一次。

數字鍵盤加號: 瀏覽游標全部報讀
利用瀏覽游標將目前物件內所在位置往下讀出直到結尾（附註：編輯游標並沒有隨之移動）。

NVDA+數字鍵盤加號: 瀏覽目標的所有物件
從目前物件開始依流程順序讀出所有的物件。

瀏覽報讀物件

數字鍵盤1: 讀出瀏覽游標前一個字元
瀏覽游標移到前一個字元並將其讀出來。

Shift+數字鍵盤1: 瀏覽游標移到行首
將瀏覽游標移到行首。

數字鍵盤2: 讀出瀏覽游標所在的字元
讀出瀏覽游標所在的字元，按二下則讀出所在字元的內碼，或在某些有支援的中文語音引擎可讀出字詞來。

數字鍵盤3: 讀出瀏覽游標的下一個字元
瀏覽游標移到下一個字元並將其讀出來。

Shift+數字鍵盤3: 瀏覽游標移到行尾
將瀏覽游標移到行尾。

數字鍵盤4: 讀出瀏覽游標前一個字詞
瀏覽游標移到前一個字詞。並將其讀出來

數字鍵盤5: 讀出瀏覽游標所在的字詞
讀出瀏覽游標所在的一個字詞，按二下則拼出該字的字元（附註：此功能適用於英文，中文則無差異）。

數字鍵盤6: 讀出瀏覽游標下個字詞
瀏覽游標移到下一個字詞並將其讀出來。

數字鍵盤7: 讀出瀏覽游標上一行
瀏覽游標移到上一行並將其讀出來。

Shift+數字鍵盤7: 瀏覽游標移到第一行
將瀏覽游標移到此文件第一行的行首。
 
數字鍵盤8: 讀出瀏覽游標所在行
讀出瀏覽游標所在的一行文字。

數字鍵盤9: 讀出瀏覽游標下一行
瀏覽游標移至下一行並將其讀出來。

Shift+數字鍵盤9: 瀏覽游標移到行尾
將瀏覽游標移到此文件的最後一行。

NVDA+F9: 標記複製文字的開始位置
利用瀏覽游標標記起點搭配下一個指令 NVDA+f10，以便將區域內的文字複製到剪貼簿。

NVDA+F10: 複製標記區的文字到剪貼簿
將前已利用瀏覽游標標記的開始位置到目前瀏覽游標所在位置之間的文字擷取並複製到剪貼簿內（附註：需與前一個指令NVDA+f9 搭配使用）。

NVDA+c: 讀出剪貼簿文字內容
讀出剪貼簿內的文字內容，如果剪貼簿的文字太多雖然程式不會一一都說出來，但會讓你知道共有多少個字元。


虛擬緩充按鍵指令
適用於  Firefox 3 （有些指令在 Internet Explorer 及 Firefox 2 可適用）

NVDA+空格鍵: 切換瀏覽模式
在網頁視窗上，切換瀏覽模式和編輯模式（又稱表單模式，在此模式下方可輸入字元）。

NVDA+Control+f: 尋找字串
在目前頁面尋找文字，按鍵後可輸入要找尋的字串，然後再按 enter 鍵。

NVDA+F3: 往下搜尋
在目前頁面向下找尋上一次輸入的字串。

NVDA+shift+F3: 往上搜尋
在目前頁面向上找尋上一次輸入的字串。

NVDA+F7: 列出元件清單
顯示一個對話方塊列出目前頁面所有的網頁元件，包括：連結項目、標題及地標等。

NVDA+F5: 更新虛擬緩衝頁面
將虛擬緩充區的資料更新為最新的頁面資料。

NVDA+v: 切換頁面排版方式
在瀏覽模式下，切換虛擬緩衝區內文字排版的方式，是否與實際螢幕編排的方式一致。


虛擬緩充物件的快速鍵
